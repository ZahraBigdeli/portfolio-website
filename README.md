# 🌐 Portfolio Website

A modern, responsive **Portfolio Website** built with **HTML, CSS, JavaScript, Bootstrap (MDB)** and **AOS Animation Library**. This project is designed to showcase a developer's skills, experience, education, projects, and contact information in a clean and professional way.

---

## ✨ Features

- 📱 Fully Responsive Design
- 🎨 Modern and Clean UI
- ⚡ Smooth Scroll Navigation
- 🎭 AOS Scroll Animations
- 👤 Hero Section
- 📝 About Me Section
- 💻 Professional Skills with Progress Bars
- 💼 Work Experience Timeline
- 🎓 Education Timeline
- 📂 Portfolio Projects
- ⭐ References Section
- 📧 Contact Form
- 🌐 Social Media Links
- 📄 Download CV Button

---

## 🛠 Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)
- Bootstrap 5 (MDB)
- Font Awesome
- AOS (Animate On Scroll)
